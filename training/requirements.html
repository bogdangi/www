General

  • Understanding of subversion is a MUST (read svn book)
  • When select what to read, priority should always be ENGLISH one
  • Try to read code, python is clear enough

Junior Developer

Unix

  • Understand main principles and basic programs for linux(unix) (commandline)
  • Able to manage|install|cofigure system packages
  • Able to use either IDE such as Eclipse or console-like as VIM, EMACS, etc

Subversion

  • Have read svn book http://svnbook.red-bean.com/, tried to work with it

GIT

  • Have read git book http://git-scm.com/book

Python

  • Have read dive-into-python book http://diveintopython.org, did all examples, all understood
  • Completed easy test program on clean python (say class proxy)

Javascript

  • Have read http://www.w3schools.com/js/default.asp or any other reference/tutorial, did examples, all understood
  • Have read http://docs.jquery.com/Main_Page, tried some examples

Markup

  • Have read standards/tutorials/reference about HTML, CSS, able to understand basics

Buildout

  • Have read about setuptools http://pypi.python.org/pypi/setuptools, pip http://pip.openplans.org/, zc.buildout http://pypi.python.org/pypi/zc.buildout and their main concepts

Zope3

Plone
